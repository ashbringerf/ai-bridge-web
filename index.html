模型

使用帮助✕

这是什么

通过网页远程给 AI 编程助手(opencode/claude/codex)发指令，AI 帮你读改代码、跑命令。指令经 GitHub 中转，无需内网穿透。

两种执行位置

① 跑在管理员电脑A（推荐，零配置）
你什么都不用装，登录后指令在管理员电脑上跑，用管理员的内网 mify 模型和工程目录。

② 跑在我自己电脑
opencode 在你自己电脑跑、改你自己的代码，模型走管理员的内网 mify。只需在你电脑跑一行安装命令(装 relay)，登录页选此模式时会显示该命令，复制到 PowerShell 执行即可。

顶栏按钮

• 引擎：切换 opencode / claude / codex
• 模型：选当前引擎用哪个模型
• 状态：看连接状态和消息记录
• 设置：改账号/执行位置

底部

• 🎤 语音输入　• 截屏 让电脑截图回传　• 发送 发指令

注意

• 管理员电脑A 的 bridge 必须开着，否则指令会等待超时。
• 经 GitHub 中转有秒级延迟，复杂任务较慢，属正常。
